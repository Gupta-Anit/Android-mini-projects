Help for Timeriffic 1.3

What is Timeriffic?

Time: Finite, and now well managed.
Turn mute and vibrate on/off custom to your schedule.
Simple. Easy. Timeriffic.
No useless features.
Won't hurt your battery!

Example: Got class M, W, F at 8am (we're sorry for ya), use Timeriffic to:

  • un-mute your volume from 7am - 8am during the ride to class,
  • switch to vibrate at 8am,
  • and then back to full volume at 9am.
(If you're a grownup substitute "class" with "work").

What's new?

Version 1.3 adds the following new features:

  • Profiles let you group actions together and enable or disable them at once.
  • Actions can be set for given days, such as actions for the week days and others for the week-end.
A couple of default profiles are provided to get you started.

To edit a profile, use long press on a profile item or an action item. A menu will appear to let you edit, insert or delete the item.

Profile screen

The "profile list" screen is what you see when Timeriffic starts.

One this screen you can do the following:


Timeriffic Enable or Timeriffic Disabled: Enable or disable Timeriffic as a whole by using the top-left toggle.

  • Check Now: Click this button to ask Timeriffic to refresh its state. The current settings will be applied, which is useful in case you changed them manually. A brief message will appear telling you when the next change will happen.
  • The purple lines are profiles names. Each profile lets you add one or more timed actions. Click on the profile checkbox to enable or disable it. Use long-press on a profile name to display a menu that will let you change its name, delete it or add more items.
  • The black lines are timed actions. These define what action happens at the specified time, on the specified days. Click on the action to edit it. Long-press the action to display a menu that lets you edit it, delete it or add more items.
